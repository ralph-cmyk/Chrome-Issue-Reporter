1. GitHub Authentication

Create a Personal Access Token at github.com/settings/tokens/new
Required scopes: repo (for private repos) or public_repo (for public repos only)

Your token is stored securely in Chrome's sync storage and never shared.

2. Default Repository

Specify the default GitHub repository where issues will be created.

Supported formats: https://github.com/owner/repo, github.com/owner/repo, owner/repo
